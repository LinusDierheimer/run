# run

a command that starts another command detached from the shell with additonal options

## Building

### required tools

* git  
* cmake  
* make  

### steps

* `git clone https://github.com/LinusDierheimer/run.git`
* `mkdir build/`
* `cd build/`
* `cmake ..`
* `sudo make install`
* finished! execute `run -h` for overview!
